## 代理池设计
代理池主要由三部分

1. 代理获取模块
   * 通过下载免费代理页面的html，分析得出所需要的代理信息存入数据库
2. REST API模块
   * 提供最基本的REST WEB API
3. 代理检测模块
   * 遍历数据库中的代理，通过访问指定的网页测试数据库中代理是否有效
